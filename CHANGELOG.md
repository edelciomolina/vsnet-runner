# Changelog

## 1.0.0

- Initial release
- Build .NET Framework projects via MSBuild from VS Code
- Run IIS Express web projects with port conflict resolution
- Run console executables with configurable args
- Build dependencies chain support
- Environment variables injection
- Attach Visual Studio 2022 debugger via COM automation
- Three context menu commands: Run .NET, Run .NET Without Args, Run .NET With Debugger
- Nearby `.netrunner` file detection with helpful warnings
- Configurable paths for MSBuild, IIS Express and devenv
